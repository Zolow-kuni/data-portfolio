# Data Analytics Portfolio — Subham Joshi

A collection of 4 end-to-end data analytics projects demonstrating skills in 
anomaly detection, operational KPI monitoring, data validation, and trend analysis.
Built using real-world datasets from Kaggle and Python-based tools.

## Projects

1. **Fraud & Anomaly Detection** — Detects fraudulent transactions using Z-score, 
   IQR, and rule-based risk scoring on 284k real credit card records.

2. **Interactive KPI Dashboard** — Streamlit dashboard tracking 12 operational KPIs 
   across Logistics, Sales, and Operations with live anomaly alerts.

3. **Financial Data Integrity Checker** — 10-rule validation engine that audits 
   financial records and generates an integrity score and violation report.

4. **Sales & Inventory Trend Analyzer** — Identifies seasonal patterns, MoM growth, 
   and stock risk flags across 2 years of regional sales data.

## Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Streamlit | Plotly | Scipy | Kaggle API
